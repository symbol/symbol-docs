用語集

絶対総量

A method of measuring amounts of mosaics while taking into account their divisibility. It is figured out by multiplying the relative amount by 10divisibility. For example, if the mosaic has divisibility 2, 10 relative units correspond to 1000 absolute units.

アカウント

アカウントは秘密鍵によってだけ変更ができるアセットのコンテナです。アカウントには常に 2 つのキー (プライベートとパブリック) とアドレスを持ちます。 詳細 を参照してください。

アカウントリンクトランザクション

A type of transaction used to transfer an account importance score to a proxy account. This is required for all accounts that wish to activate delegated harvesting. Read more.

アカウント制限

ブロックのアナウンスや特定のアカウントへのトランザクションの受信へ対する設定可能なスマートなルールです。 詳細 を参照してください。

アグリゲートボンド (パーシャルトランザクション)

他の参加者からの署名が必要な場合、アグリゲートトランザクションは ボンデッド になります。 詳細 を参照してください。

アグリゲートコンプリート

要求されている参加者全員が署名をすると、アグリゲートトランザクションはコンプリートになります。 詳細 を参照してください。

アグリゲートトランザクション

ワンタイムの破棄可能なスマートコントラクトを作成することで、複数のトランザクションを一つのトランザクションへマージしたトランザクションタイプです。詳細 を参照してください。

Agora

The codename for a decentralized, peer-to-peer marketplace built on Symbol to enable the trading of mosaics.

エイリアス

The namespace linked to an account or mosaic using Alias transactions. An alias and its linked object can be used interchangeably when sending a transaction. Read more.

AMA

Ask Me Anything。公開質疑応答。

AML

アンチマネーロンダリング

Andromeda

The codename for a workflow product by NEM that enables users to visually architect and deploy dApps.

APAC

アジアと太平洋地域

API ノード

Nodes responsible for storing data in a readable form in MongoDB. They are also responsible for collecting the cosignatures of aggregated bonded transactions.

APR

年換算利回り

アービトラージ

トレーダーが資産をあるところで購入し、別のところで売却することで、市場間に発生する価格差から利益を得ること。

デリゲートハーベスターの自動検出

The automatic process by which Symbol servers allows accounts to register as delegated harvesters via special transfer messages.

バックランニング

To broadcast transactionA with slightly lower gas (or fees) than an already pending transactionB so that transactionA gets mined right after transactionB in the same block.

BLS

A Boneh–Lynn–Shacham signature is a cryptographic signature scheme which allows a user to verify that a signer is authentic.

BTC

ビットコイン

カタパルト

NEM 2.0 のオリジナルプロジェクト名 - 後に Symbol となりました。

CBDC

中央銀行デジタル通貨

CLI

コマンドラインインターフェイス。キーボードだけを使い、完全にターミナルコンソールで使用するプログラムのこと。Symbol にはブロックチェーンと疎通するための CLI ツールがあります。 より詳細 を参照してください。

CMC

コインマーケットキャップ。ウェブページ。

連署者

マルチシグアカウント のアカウントマネージャとして機能するアカウント。連署者はネットワークにアナウンスする前に、マルチシグアカウントトランザクションに署名する必要があります。

連署名

トランザクションを承認するための署名を提供する行為。

クロスチェーンスワップ

第三者 (交換サービスなど) を使用せずに、異なるブロックチェーン間でトークンを取引できる Symbol の組み込み機能。 詳細 を参照してください。

CSD

Central Securities Deposit.

DAO

分散型自立型組織

dApp

Decentralized Application. An application that runs on a blockchain instead of a single computer. The term is slightly abused so, in a more general sense, it also means any application which makes use of a blockchain.

DDH

Decisional Diffie-Hellman.

DD

デューディリジェンス

Deadline

A time window for a transaction to be accepted before it reaches its expiration. The transaction is eliminated when the deadline is reached and all the nodes reject the transaction. Read more.

DeFi

Decentralized Finance, as opposed to Traditional Finance (TradFi).

デリゲートハーベスティング

インポータンススコアを新しいプロキシアカウントに委任することにより、ユーザがノードをローカルで実行しなくても報酬を受け取ることができるハーベスト方法。 詳細 を参照してください。

DEX

Decentralized Exchange.

DID

Decentralized ID.

Divisibility

モザイクの小数分割を可能にするプロパティ。可分性はモザイクを分割できる小数点以下の桁数を指定します。

DTC

Direct To Consumer, i.e. mass market.

Duration

Length of time measured in ブロック で測定された時間の長さ。Symbol ブロックチェーンの各ブロックはおよそ30秒ごとにハーベストされます。

E2E

End-To-End.

実行手数料

The fee to be paid for a transaction. Calculated by reading the fee multiplier from the block in which the transaction got confirmed and multiplying it by the size of the transaction.

EMEA

Europe, Middle-East and Africa.

ERC

Ethereum Request for Comment. Commonly utilized to refer to a token standard on the EVM (such as ERC-20, ERC-721, ERC-1155).

ETH

Ethereum.

EVM

Ethereum Virtual Machine.

手数料乗数

A multiplier used to calculate the effective fee of each transaction contained within a block.

FFT

Fast Fourier Transform.

Flashbots

A research and development organization working on mitigating the negative effects of MEV extraction techniques.

Frontrunning

In context of cryptocurrencies, Trying to include your transaction in front of some other transaction. This is more important in case of DeFi markets, where gains can be made from front-running.

グローバル制限

Network-wide rules that determine whether accounts will be able to send or receive a specific mosaic (with Mosaic Restrictions enabled).

ハーベスタ

The account that harvests a block. The account is rewarded with the transaction fees added in the block and the inflation tokens generated.

ハーベスティング受益者

ノードオペレーターが指定する、ブロック報酬の一部を共有するアカウント。

ハーベスティング

Symbol ブロックチェーンにおいての新しいブロックを作成するプロセスです。 詳細 を参照してください。

ハッシュロックトランザクション

A type of transaction which locks funds for a certain amount of blocks. This transaction is required before announcing an Aggregate Bonded Transaction. When the associated Aggregate Transaction is complete, the locked funds are returned to the original account. Read more.

HTLC

Hashed Time Lock Contract. A protocol which creates a trustless environment for the decentralized exchange of assets. It guarantees that a swap will take place if all the participants agree. On the other hand, if some of them decide not to conclude the process, each participant will receive their locked funds back.

Hermes

Messenger of the Gods. Codename for Symbol's next wallet project.

ICO

Initial Coin Offering.

インポータンススコア

A value calculated by the PoS+ algorithm based on three factors that determine the probability that an account has to harvest a block.

インフレーション

Network configured increase in currency supply per block. The mosaics created due to inflation are included in the block reward. Read more.

IP

Intellectual Property.

IRS

Internal Revenue Service. Who you pay your taxes to if you live in the US or are an American citizen.

IVC

Incrementally Verifiable Computations.

Kairos

From Ancient Greek: "The right, critical, or opportune moment". The codename for a collectible card game, built on top of Symbol.

KYV

Know Your Customer.

LATAM

Latin America (Central and South America).

ローカルハーベスティング

ローカルノードで ハーベスティング を実行する方法

M&A

Mergers & Acquisitions (NOT a chocolate candy).

最大手数料

トランザクションが承認されるために許可する、送信者によって支払われる、最大 手数料 。

マークル木

A structure of nodes labeled by hashes. It is a data validation technique used by Symbol to store large data sets associated with a block that cannot be retrieved directly from the block header. It allows light clients to verify if an element (e.g. transaction, receipt statement) exists without demanding the entire ledger history.

メッセージ

トランザクション にテキスト文字列を添付する機能

メタデータ

アカウントやモザイクに添付される追加情報です。 詳細 を参照してください。

MEV

Miner-Extractable Value - process of reorganising transactions inside a block by miners, to gain something (might be covered by secret contract).

最小承認数

Number of cosignatories required for the multisignature account to execute a transaction.

最小削除数

Number of cosignatories required to remove a cosignatory from a multisignature account.

モザイク制限

モザイク作成者にアカウントがアセットを使った取引を制御できるようにする機能。これは作成時に明示的に有効化された制限可能プロパティを持つモザイクにのみ影響します。 詳細 を参照してください。

Mosaic

A representation of any kind of asset on Symbol (fungible or non-fungible). Read more.

マルチレベルマルチシグネチャアカウント

An advanced built-in feature of Symbol that allows multisignature accounts to be cosigners for other multisignature accounts, creating multiple layers of cosignatories. Multi-level multisignature accounts add “AND/OR” logic to multi-signature transactions.

マルチシグネチャ (マルチシグ) アカウント

アクション / 転送の開始に (連署者からの) 追加署名が必要なアカウントです。 詳細 を参照してください。

ネームスペース

Symbol ブロックチェーン上の一意なドメインスペース。Symbol アカウントまたはモザイクにリンクできます。インターネットドメインと同様に機能します。 詳細 を参照してください。

NAM

North America.

Nanowallet

Desktop NIS1 wallet created by the NEM Foundation.

NEM

The New Economy Movement.

ネットワーク手数料シンク

An account defined by the network operator that will receive a percentage of the harvesting rewards.

NFT

A non-fungible token - a way to represent anything as an Ethereum-based asset.

NGL

NEM Group Limited.

NIS1

The first version of NEM's blockchain node that operates the public mainnet with the native currency XEM. First launched on March 31, 2015.

ノード拒否

The act by which the Symbol network will prevent communication with a malicious remote node and reject incoming connections from it.

ノードレピュテーション

A measure of trust that the Symbol network determines for each specific node. The network's trust for a node increases with each successful interaction, and decreases for each failed attempt of communication.

NODE_URL

The URL of the node you want to use to access the network. All nodes should return the same information so it is not critical which one you use.

Use the Statistics Service (or the Testnet Statistics Service) to retrieve a list of nodes, choose one and use its restGatewayUrl as your NODE_URL (Including the port number).

NSL

NEM Software Limited.

オフチェーン

Realm outside of the blockchain. Off-chain activity does not directly reflect on the blockchain.

Optimistic Rollups

An Ethereum layer 2 scaling solution. Optimistic Rollups.

PDHU

Persistent Delegated Harvesting Unlocking. A feature that enables delegated harvesters to preserve their status despite connectivity problems of nodes. With PDHU, if a node experiences turbulence and reboots, the existing delegated harvesters will automatically reconnect when the node is back online.

Peer ノード

Nodes that facilitate the blockchain process by verifying transactions and blocks, running the consensus algorithm, creating new blocks, and propagating the changes through the network.

Perseus

The codename for a end-to-end simulation platform by NEM that allows backtesting of network upgrades. Will launch with Symbol support but other blockchains can be added.

PoC

Proof of Concept (NOT a consensus protocol).

PoI

Proof of Importance. The consensus protocol used by NIS1. Similar to PoS but measuring an account's activity besides its stake.

PoS

Proof of Stake. A consensus protocol.

PoS+

Proof-of-Stake Plus. Symbol's consensus mechanism. It is a modified PoS algorithm which considers users' activity in the network in addition to their network stakes. The chance that accounts will have to harvest a block is calculated through their importance scores. Read more.

PoW

Proof of Work. A consensus protocol.

秘密鍵

Cryptographic key that gives ultimate control over an account and its assets, and must thus be kept secret. It is paired with the public key in the key pair system.

公開鍵

広く播種させることのできるキーペアの公開識別子です。対となる秘密鍵でトランザクションが署名されていることを証明します。公開鍵は秘密鍵から暗号的に導出できます。

レシート

Record of proof for every hidden change on the blockchain. The collection of receipts are hashed into a merkle tree and linked to a block. Read more.

リファレンスモザイク

A mosaic selected by the mosaic creator to define mosaic restrictions that depend directly on the selected mosaic's global restrictions.

Relative Amount

A method of measuring amounts of mosaics without accounting for their divisibility. It is figured out by multiplying the absolute amount by 10divisibility. For example, if the mosaic has divisibility 2, 10 relative units correspond to 1000 absolute units.

レンタル手数料

Fees required to register a namespace or extend its duration. The default namespace rental fees are configurable per network, but the network dynamically adjusts the namespace rental fees over time.

ロールバック

以前に承認された ブロック を元に戻す行為。

Rug Pull

A malicious maneuver where cryptocurrency developers abandon a project and run off with the funds.

Sandwitch

A type of front-running technique that's popular in DeFi. To make a sandwich, you find a pending transaction in the network and then try to surround the network by placing one order just before the transaction (front-running) and one order just after it (back-running).

SDK

Software Development Kit. A Software library used to simplify creating applications for a given platform. Read more about Symbol's SDK.

シークレットロックトランザクション

A type of transaction between two accounts where the mosaics remain locked until the recipient presents a valid SecretProofTransaction. Otherwise, the funds are returned to the sender. Read SecretLockTransaction.

シークレットプルーフトランザクション

A type of transaction through which the recipient of a SecretLockTransaction provides proof and unlocks the mosaics. Read SecretLockTransaction.

Sharding

An Ethereum scaling solution. Sharding.

スパムスロットル

大量の未確認のトランザクションでスパムされることに対して Symbol ネットワークの保護を提供する機能。

サブネームスペース

A unique domain on the Symbol blockchain that is a part of a larger domain under the namespace hierarchy. Subnamespaces can only exist in conjunction with a root namespace.

SXDH

Symmetric External Diffie-Hellman.

Symbol 拡張

Plugins that can be added to the Symbol's protocol to extend its capabilities. Allows developers to introduce different ways to alter the chain's state via transactions without modifying the core engine or disrupting other features. Read more.

TLC

Tender Loving Care.

TLS

Symbol ブロックチェーン上でピア間の通信を暗号化するために使用されるセキュリティプロトコル。

Token

A representation of a digital asset.

TPS

Transactions Per Second.

トランザクションプラグイン

ネットワークがサポートするトランザクション種類を決定するプラグインのセット。

転送トランザクション

モザイクとメッセージの送信に使用される最も基本的なトランザクション。 詳細 を参照してください。

USP

Unique Selling Proposition or Unique Selling Point. A characteristic of a product that can be used in advertising to differentiate it from its competitors.

VPS

Virtual Private Server. A virtual machine typically hosted on a Data Center which you can access remotely and treat as if it was your own physical machine.

VRF

VRF stands for verifiable random function. All potential harvesting accounts must link to a second public key by announcing a VrfKeyLinkTransaction. The key linked is then used to randomize block production and leader/participant selection.

XEM

NEM's NIS1 blockchain native currency.

XYM

Symbol ブロックチェーンの基軸通貨です。
