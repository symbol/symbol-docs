アセットの管理

デスクトップウォレット

アカウント、モザイク、名前空間を管理し、トランザクションを発行するための Symbol 用クロスプラットフォームクライアント。Symbol デスクトップウォレットは Mac, Windows と Web アプリケーションが提供されています。

Catapult REST (API)

catapult-rest REST パッケージは HTTP と WebSockets をブロックチェーン上で読み書きアクションが実行できるように併合します。

ソフトウェア開発キット (SDK)

主要なソフトウェア開発ツールは Symbol コンポーネント、このような追加ツール、ライブラリ、アプリケーションなどを作成するためのものです。高度な抽象化により、開発者は特定の API 詳細ではなく製品に集中できます。

システム管理者

Catapult Client

Symbol ベースのネットワークは、トラストレスな、高性能で安全なブロックチェーンプラットフォームを提供するためにノードに依存しています。

これらのノードは2015年から稼働し続けてきた、従来 JAVA で書かれていた NEM 分散台帳を C++ でリライトした catapult-client ソフトウェアを使用してデプロイされます。

Symbol Bootstrap

symbol-bootstrap パッケージは、コマンドの実行だけで、テストまたは開発目的のネットワークをデプロイするために必要なセットアップスクリプトを含んでいます。
